# Diabetic-Foot-Ulcer-DFU-Detection-Classification-Deep-Learning-DFUC2024

Developed a DFU image classification model using transfer learning, improving DFU severity/infection prediction on DFUC2024 dataset. 
Trained EfficientNet/ ResNet-based DFU classifiers with learning-rate scheduling, early stopping, and hyperparameter tuning. 
Enhanced performance using augmentation + class imbalance strategies (focal loss/class weights), improving minority-class F1. 
Performed evaluation using F1-score, AUC, confusion matrix, and implemented Grad-CAM for interpretability.
